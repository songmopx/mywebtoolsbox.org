# 个人网站导航 | Personal Navigation

一个现代化的个人网站导航页面，帮助你整理和管理喜欢的网站链接。支持自定义分类、多语言切换，并具有优秀的响应式设计。

## ✨ 功能特性

### 🏠 核心功能
- **网站管理**：添加、编辑、删除网站链接
- **分类管理**：创建自定义分类，支持图标和颜色配置
- **智能搜索**：快速搜索网站名称和描述
- **多语言支持**：中英文界面切换
- **响应式设计**：完美适配桌面端和移动端

### 🎨 界面设计
- 现代化的渐变背景设计
- 精美的卡片式布局
- 流畅的动画效果
- 网站图标自动获取
- 直观的操作界面

### 🔧 技术特性
- **SEO优化**：完整的meta标签配置
- **本地存储**：数据自动保存到浏览器
- **无依赖**：纯HTML+CSS+JavaScript实现
- **快速加载**：CDN资源加载
- **安全访问**：新窗口打开外部链接

## 🚀 快速开始

### 安装
1. 下载或克隆项目到本地
2. 用浏览器打开 `index.html` 文件
3. 开始使用！

### 使用方法

#### 添加分类
1. 点击右上角的"添加分类"按钮
2. 填写分类名称、选择图标和颜色
3. 点击保存即可创建新分类

#### 添加网站
1. 点击右上角的"添加网站"按钮
2. 填写网站名称、URL地址和描述
3. 选择所属分类
4. 点击保存即可添加网站

#### 搜索网站
- 在搜索框中输入关键词
- 自动匹配网站名称和描述
- 实时显示搜索结果

#### 语言切换
- 点击右上角的语言切换按钮
- 支持中文和英文界面

## 🛠️ 自定义配置

### 修改默认数据
编辑 `js/app.js` 文件中的 `addDefaultCategories()` 方法：

```javascript
const defaultCategories = [
    { id: 1, name: '你的分类', nameEn: 'Your Category', icon: 'fas fa-star', color: 'blue' }
];

const defaultWebsites = [
    { id: 1, name: '网站名称', url: 'https://example.com', categoryId: 1, description: '描述' }
];
```

### 添加新的颜色主题
在分类颜色选项中添加新颜色：

```javascript
// 在 renderCategory 方法中添加新的颜色类
const colorClasses = {
    // 添加你的颜色
    orange: 'from-orange-500 to-orange-600'
};
```
